# Umbraco-8-Bulma-Starter-Kit
Starter Kit for Umbraco V8 based on the Bulma Css framework.

It has the following content templates:

* Homepage
* Products listing page
* Product detail page
* People listing page
* Generic content page
* Blog listing page
* Blog detail page
* Contact page
